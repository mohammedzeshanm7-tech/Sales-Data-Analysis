# 📊 Sales Data Analysis Project

## 📌 Project Overview
This project analyzes sales data using Python and Pandas.  
The goal is to understand total sales performance across different countries.

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- CSV Dataset

## 📈 Key Analysis Performed
- Loaded and explored sales dataset
- Grouped data by country
- Calculated total sales per country
- Created bar chart visualization

## 📊 Output
The project generates a bar chart showing total sales by country.
